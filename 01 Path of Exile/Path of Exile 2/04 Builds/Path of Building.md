## 1. Getting Started

### What is Path of Building?
Path of Building is a desktop application that allows you to theoretically plan, build, and optimize Path of Exile 2 characters. It calculates damage, defenses, and other statistics before you even log into the game.

### Installation
- Download from: https://pathofbuilding.community/
- Extract to your preferred location
- Run `Path of Building.exe`
- **Optional:** Add to your PATH for quick terminal access

### Initial Setup
- Select **PoE2** as your game version (important for correct mechanics)
- Update data files when prompted
- Configure performance settings if needed

---

## 2. Basic Interface Navigation

### Main Layout
- **Left Panel:** Character info, skills, items, passive tree
- **Center Panel:** Passive tree view (clickable)
- **Right Panel:** Stats, DPS, calculations, survivability
- **Top Bar:** Menu, save/load, import/export options

### Key Tabs
- **Skills:** Add and link gems
- **Items:** Configure equipment
- **Config:** Set up conditions (buffs, debuffs, combat state)
- **Calcs:** View detailed calculations
- **Tree:** Interact with passive tree

---

## 3. Character Setup

### Class Selection
1. Click **Character** → Select your class
2. Choose your starting point on the passive tree
3. Set your level (default: 1, max adjustable)

### Level Management
- Manually set your character level
- Passive points scale with level
- Allocate points efficiently on the tree

### Ascendancy Setup
- Go to **Character** menu
- Select your Ascendancy class
- Configure Ascendancy passive choices (if available in PoE2)

## 4. Build Configuration

### Passive Tree Planning
1. **Click nodes** on the tree to allocate passives
2. **Right-click** to unallocate
3. **Hover** over nodes to see stat details
4. **Search function** to find specific passives
5. **Path lines** show your pathing efficiency

### Keystones & Notables
- Filter the tree to highlight Keystone passives
- Plan major defensive or offensive keystones early
- Review life/ES wheels for survivability

### Pathing Efficiency
- Path of Building shows shortest path between nodes
- Aim to maximize value per point spent
- Balance offense and defense

---

## 5. Skill Gems & Linking

### Adding Skills
1. Go to **Skills** tab
2. Click **"+"** to add a skill
3. Search for gem name or browse categories
4. Select quality level (typically 20% for optimal)

### Gem Linking
1. Click your main skill gem
2. Click **"+"** to add support gems
3. Add up to 5 support gems (6-link items)
4. **Ensure color requirements match** your item sockets

### Gem Quality
- **20% Quality:** Standard (from currency)
- **Different quality types:** Some gems have special quality effects
- Adjust in-app for accurate damage calculations

### Support Gem Strategy
- Prioritize **more multiplier** supports
- Add **utility supports** (faster cast, elemental penetration, etc.)
- Check for gem stat scaling (e.g., spell critical, attack speed)

---

## 6. Item Management

### Adding Equipment
1. Go to **Items** tab
2. Click each slot (weapon, armor, ring, etc.)
3. Select an item or enter custom stats
4. Adjust socket colors if needed

### Item Database
- Path of Building has a built-in item database
- Search by item name, type, or requirement
- Filter by rarity (unique, rare, magic, normal)

### Custom Items
- Create custom rare/magic items
- Input stats manually for theory-crafting
- Useful for planning gear upgrades

### Flask Management
- Configure active flasks in the config
- Select charges, effects, and durations
- Include buff effects in calculations

## 7. Configuration & Conditions

### Buff/Debuff Setup
1. Open **Config** tab
2. Toggle relevant conditions:
   - **You are:** Leeching, on low life, fortified, etc.
   - **Enemy:** Cursed, shocked, poisoned, etc.
   - **Buffs:** Auras, banners, cries active

### Combat Assumptions
- **Full Life / Low Life:** Changes some passive effects
- **Leech Active:** For sustain calculations
- **Critical Strike Conditions:** Frenzy charges, precision, etc.

### Enemy Configuration
- Select enemy type (normal, rare, unique)
- Set resistances and armor
- Configure curse effectiveness

## 8. DPS Calculation & Damage Analysis

### Main DPS Metrics
- **Hit Damage:** Single hit damage value
- **DPS (Damage Per Second):** Sustained damage output
- **Average Damage:** Typical hit value

### Understanding the Calcs Tab
1. Open **Calcs** for detailed breakdowns
2. View damage calculation steps
3. Identify scaling sources
4. See hit chance and critical information

### Optimizing Damage
- Compare support gem changes
- Test different passives
- Evaluate item upgrades
- Analyze scaling with attributes

### Special Mechanics
- **Poison/Bleed Damage:** Separate calculations
- **Elemental Damage:** Conversion, penetration, exposure
- **Critical Strikes:** Multiplier and chance
- **Ailments:** Application rates and damage

---

## 9. Survivability & Defense

### Life/ES Pool
- View total effective health (in **Calcs**)
- Plan for adequate life % on gear
- Balance offensive and defensive scaling

### Resistances
- Aim for **cap (75%)** on all resistances
- Account for curse effects
- Plan for uncapped scenarios

### Armor & Block
- Calculate mitigation from armor
- Plan block percentage if using shields
- Compare to evasion builds

### Defensive Layers
- **Life Recovery:** Leech rate, flask healing
- **Recovery Speed:** Regen, recovery per second
- **Avoidance:** Dodge, evasion chance
- **Tankiness:** Fortify, endurance charges

### Damage Taken Modifiers
- Configure in **Config** (taken as extra elemental damage, etc.)
- Impacts effective health calculations
- Critical for high-damage content planning

## 10. Optimization & Comparisons

### A/B Testing
1. **Save your build** with a version name
2. **Make changes** (swap gem, respec passive)
3. **Compare stats** side by side
4. Use **Import** to reload previous versions

### Optimization Tips
- Test passive swaps systematically
- Evaluate small gear upgrades
- Calculate cost/benefit of each change
- Prioritize efficient improvements

### Benchmarking
- Note your DPS against different enemies
- Track survivability metrics
- Identify weak points in your setup
- Plan progression goals

## 11. Saving, Loading & Sharing

### Saving Builds
1. **Ctrl+S** or **File** → **Save**
2. Name your build (e.g., "Fire Blaster v2")
3. Builds auto-save to local directory

### Loading Builds
- **File** → **Recent Builds** for quick access
- Or manually browse saved files
- Organize by class/type for easy navigation

### Exporting Builds
1. **Copy skill tree link** (shareable URL)
2. **Export to text** for notes/sharing
3. **Share pastebin link** with others

### Importing
- **Paste item rarity string** from in-game
- **Import builds** from text exports
- **Load skill tree** from links

## 13. Troubleshooting & Tips

### Common Issues

**Build not calculating correctly?**
- Verify gem colors match socket colors
- Check that support gems are compatible
- Confirm item requirements are met

**DPS seems too high/low?**
- Check your **Config** settings (buffs/debuffs)
- Review **Calcs** tab for scaling sources
- Compare against in-game damage

**Can't find a gem/item?**
- Ensure you're using the **PoE2** data version
- Update data files from the app
- Try searching by partial name

**Passive tree looks wrong?**
- Verify correct character class selected
- Check that you're viewing PoE2 tree
- Reset tree if corrupted (rare)

### Pro Tips
1. **Save often** - Create versioned builds for comparison
2. **Use Config liberally** - Conditions dramatically change results
3. **Check Calcs** - Understand where damage comes from
4. **Plan defensively** - Don't optimize only for damage
5. **Test progressively** - Evaluate changes one at a time
6. **Keep notes** - Document your build philosophy in [[PoE2 Build Notes]]
